Open Source Discord Bot

Smart tools for
League of Legends

A free Discord bot that provides scouting reports, team balancing for custom games, match history analysis, and real-time game insights for your friend group.

Features

/scout Opponent Scouting

Auto-detects opponents in your live game and generates a scouting report with rank, champion stats, win rates, and threat highlights.

/teams Team Scrambler

Fetches player ranks and creates skill-balanced teams for custom games. Supports random and balanced modes with optional position assignment.

/history Match History

View recent match performance including win streaks, champion stats, and KDA for any player.

/matchup Matchup Data

Compare head-to-head champion statistics to help with draft decisions and lane preparation.

/multisearch Multi-Search

Look up all 5 teammates at once during champ select. See everyone's rank, win rate, and champion pool in one embed.

/meta Live Meta Data

Get current champion builds, runes, skill order, win rates, and tier rankings sourced from live game data.

Riot API Usage

OpenLoL uses the Riot Games API to provide accurate, real-time player and game data. All access is read-only. No gameplay is automated or modified.

account-v1 Resolve Riot IDs to PUUIDs for player lookup
summoner-v1 Summoner profile data and account info
league-v4 Ranked tier, division, LP, and win/loss record
match-v5 Match history, post-game stats, and KDA
spectator-v5 Live game detection for linked users
champion-v3 Champion rotation and static game data

Privacy Policy

What Data We Access

OpenLoL accesses only publicly available game data through the Riot Games API:

  • Summoner names, ranks, and match history (public profile data)
  • Live game participant information via Spectator API
  • Champion and game metadata

What Data We Store

  • Discord-to-Riot ID links: When a user runs /link, we store the mapping between their Discord user ID and Riot ID. This can be removed at any time with /unlink.
  • No match data is permanently stored. Game data is fetched on-demand and discarded after the Discord message is sent.
  • No personal information beyond Discord user IDs and voluntarily linked Riot IDs is collected.

Data Sharing

We do not sell, share, or transfer any user data to third parties. All data is used exclusively to provide bot functionality within Discord servers where the bot is installed.

Data Retention

Discord-to-Riot ID mappings are stored for as long as the link is active. Users may remove their linked account at any time. No other user data is retained.

Third-Party Services

  • Riot Games API: Used to retrieve game data. Subject to Riot's Developer Policies.
  • Discord API: Used for bot functionality. Subject to Discord's Terms of Service.

Contact

For privacy concerns or data removal requests, open an issue on our GitHub repository.
